<p align="center">
  <strong>MWA301</strong> is a repository focused on <strong>web application hacking</strong>, practical security learning, and hands-on vulnerability assessment concepts.
  It is designed to support structured learning, offensive security practice, lab content, and curated module resources for modern web application testing.
</p>

---

## Overview

This repository serves as a central workspace for:

* Web application security notes and module resources
* Vulnerability research and walkthroughs
* Practical lab exercises and proof-of-concept material
* Training support content for learners and delegates
* Security testing references, scripts, and assessment assets

The goal is to present technical content in a way that is both **structured** and **visually engaging**, while remaining useful for classroom delivery, independent study, and lab-based practice.

## Learning Outcomes

By using this repository, learners should be able to:

* Understand how web applications process structured input
* Explain how XML parsing features can create security risk
* Identify unsafe parser behavior linked to XXE
* Build or test controlled lab scenarios safely
* Recommend secure mitigations and defense-in-depth controls
* Translate technical findings into professional security explanations
